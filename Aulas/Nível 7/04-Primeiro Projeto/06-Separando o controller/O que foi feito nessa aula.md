# Conteúdo da aula

Nessa aula, foi criado o primeiro controller da API. Nele, foi separada a responsabilidade de executar a lógica principal da rota. Além disso, foi criado um arquivo para rotas, onde foi definido o end-point para a API. A API, por sua vez, permite buscar lista de tickets, criar um novo tickets e deletar um tickets.

Controller basicamente servirá para executar algo quando nossa rota for chamada. Sendo uma boa prática separar responsabilidades quando se tem muitas responsabilidades
