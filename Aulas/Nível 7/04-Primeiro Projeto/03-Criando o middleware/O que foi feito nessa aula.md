# Conteúdo da aula

** Criar o middleware para identifiar o conteúdo que vem pelo body das requisições. **

Nessa aula, foi criado um middleware para identificar o conteúdo que vem pelo body das requisições. Isso foi feito para que a API possa ler dados no formato JSON e, com base nessa informação, possa processar as requisições de forma adequada. Além disso, o middleware também foi programado para que possa retornar erros caso a requisição contenha dados inválidos.
