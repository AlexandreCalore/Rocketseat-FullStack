# Conteúdo da aula

Nessa aula, foi criado o primeiro projeto em Node.js. Nele, foi criado um middleware para identificar o conteúdo que vem pelo body das requisições. Além disso, foi criado um arquivo para rotas, onde foi definido o end-point para a API. A API, por sua vez, permite buscar lista de tickets, criar um novo tickets e deletar um tickets.
