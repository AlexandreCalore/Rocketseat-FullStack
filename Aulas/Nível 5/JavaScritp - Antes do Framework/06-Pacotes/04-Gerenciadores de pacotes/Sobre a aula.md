# Gerenciador de pacotes

É uma ferramenta que facilita a instalação, atualização e depêndencias de um projeto.

O gerenciador de pacotes lida com a resolução de dependências, garantindo que as bibliotecas necessárias estejam instaladas corretamente com versões compatíveis.

## NPM

NPM (Node Package Manager) é um gerenciador de pacotes mais populares em JS.

Usado principalmente no ecossistema do Node.js, mas também é amplamente utilizado em projetos front-end web.

O NPM permite instalar e gerenciar pacotes de JS. Também permite executar scripts e bibliotecas instaladas.

## Dependências e Ambientes

**Produção**

Inclui as dependências (pacotes/bibliotecas) necessárias para a execução do aplicativo em produção (por exemplo transpiladores).
