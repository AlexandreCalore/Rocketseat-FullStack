Nesta aula, foi estudado a diferença entre npm (node package management) e npx (node package execute). O npm é um gerenciador de pacotes que permite ao desenvolvedor instalar e gerenciar pacotes para o Node.js. Já o npx é um executador de pacotes que permite ao desenvolvedor executar os pacotes do Node.js sem a necessidade de instalar previamente. Além disso, foi estudado como executar o TypeScript com o TSX e como criar um projeto com o TypeScript.
